# AI Agent Flutter v1.31

Версия v1.14 усиливает восстановление C++/CMake сборки после анализа логов v1.13.

Главные изменения:

- `run_command` получил необязательный параметр `cwd` для запуска команд в подпапке проекта.
- `run_tests` сам ищет `.cpp/.cc/.cxx` файлы в корне и подпапках проекта.
- Для C++ задач `run_tests` может создать или исправить минимальный `CMakeLists.txt` только для сборки уже созданного моделью исходника.
- Если CMake сообщает `does not appear to contain CMakeLists.txt`, `Cannot find source file`, `No SOURCES given to target`, агент считает это ошибкой структуры сборки, а не ошибкой логики кода.
- После нескольких неудачных ответов модели агент делает ограниченное автоматическое восстановление через `run_tests` и возвращает результат модели.
- На Windows, если нет `g++`, но есть CMake/Visual Studio, `run_tests` использует CMake/Visual Studio для сборки.
- Код предметной задачи по-прежнему должна писать модель; встроенных C++ решений нейросети не добавлено.

Проверка после распаковки:

```bat
flutter clean
flutter pub get
flutter analyze
build_all.bat
```


## Обновление v1.16: защита служебной папки .cppagent

`.cppagent` теперь считается строго служебной папкой агента. Модель не может создавать там исходники, CMakeLists.txt, README, тесты, ассеты или другие рабочие файлы пользователя через write_file/append_file/replace_text/make_dir/copy_path/move_path/delete_path. Для пользовательского кода агент должен использовать корень проекта или обычные папки `src`, `include`, `tests`, `data`, `assets`, `build_scripts`.

## Обновление v1.27: Android, Web, dist, база знаний

- Добавлен ползунок масштаба интерфейса и автоматический компактный режим для Android.
- Консоль на Android запускает команды через `/system/bin/sh` в собственной среде приложения.
- Web-вкладка получила Android-разрешения на интернет и cleartext HTTP для локальных адресов.
- Сборки копируются в `dist` с именем программы, версии, ОС и архитектуры.
- Добавлены `web_deep_fetch`, `filesystem_search`, `knowledge_store`, `knowledge_search`.
- База знаний доступна в настройках, поддерживает экспорт/импорт ZIP/JSONL.
- Добавлена строгая проверка ФИО в задачах поиска людей.


## Обновление v1.31: качество поиска, права проекта, Android-консоль и ссылки
