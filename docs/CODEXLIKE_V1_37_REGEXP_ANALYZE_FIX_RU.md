# AI Agent v1.37 — исправление Dart RegExp для analyze/build

Исправлена ошибка компиляции в `lib/main.dart` около строк 6995–7003.

Причина: raw-строки `r"..."` содержали неэкранируемые двойные кавычки внутри шаблонов регулярных выражений (`["']`), поэтому Dart разрывал строку и воспринимал часть выражения как список.

Исправление: проблемные регулярные выражения заменены на raw triple-quoted строки `r'''...'''`.

Дополнительно версия build-скриптов обновлена до v1.37.
