# v1.21 — консоль, быстрый запуск и проверка артефактов сборки

## Главное

1. Команда сборки теперь не считается успешной только по `EXIT_CODE: 0`, если в командной строке указан выходной файл через `-o build/app.exe`, `-o build\\app.exe` или `/Fe:...`.
2. После такой команды агент проверяет реальное наличие файла и добавляет в результат:
   - `BUILD_ARTIFACT_OK: ...`
   - или `BUILD_ARTIFACT_MISSING: ...`
3. Если артефакт отсутствует, результат команды принудительно считается ошибкой `EXIT_CODE: 2`, чтобы агент не писал пользователю, что программа собрана.
4. Если артефакт найден, создаётся кнопка быстрого запуска в консоли.

## Вкладка «Файлы»

- Исполняемые файлы можно запускать из контекстного меню.
- Запуск открывает новую вкладку «Консоль» и выполняет выбранную программу.
- Для запуска также создаётся кнопка быстрого запуска.

## Вкладка «Консоль»

- Ввод команд расположен внизу.
- В выводе консоли показывается только `stdout`; если stdout пустой, показывается stderr/краткая ошибка.
- Промпт отображает текущую папку в компактном виде проекта.
- ПКМ по области вывода: `Копировать`, `Очистить`.
- ПКМ по строке команды показывает меню:
  - `Выделить всё`
  - `Копировать`
  - `Вставить`
  - `История команд`
  - `Программы`
  - `Файлы из папки`
  - `Очистить`
- Для истории/программ/файлов открывается второе меню. Если строка команды уже заполнена, приложение спрашивает: добавить или заменить.

## Адаптивность

- На узком окне правая панель быстрых кнопок в «Консоль» и «Web» скрыта по умолчанию.
- Её можно открыть кнопкой в верхней строке вкладки.
