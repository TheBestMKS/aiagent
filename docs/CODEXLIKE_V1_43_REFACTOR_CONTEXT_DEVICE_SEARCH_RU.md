# AI Agent v1.46 — разбиение main.dart, контекст, файлы и поиск документов

Изменения:

- `lib/main.dart` разделён на `part`-файлы в `lib/parts/`, чтобы основной файл больше не был монолитным.
- При выполнении задачи переключение проекта ставится в очередь и не ломает активную задачу.
- Сырые повреждённые `<tool_call>` больше не показываются в диалоге; агент просит модель повторить корректный tool-call.
- Автоматическое сжатие контекста теперь учитывает только сообщения ниже последней черты «Автоматическое сжатие контекста».
- После сжатия добавляется видимый блок «Сжатый контекст» с пояснением, что история выше черты больше не отправляется модели.
- `list_device_directory` и чтение папок теперь пропускают недоступные системные папки вроде `$RECYCLE.BIN`, а не прерывают всю задачу.
- Добавлен tool `search_device_documents` для поиска документов по смысловым ключам в папке и подпапках с извлечением текста из офисных/PDF/архивных форматов best-effort.
- Текстовый редактор открывает больше форматов через общий механизм извлечения текста: txt/log/h/cpp/py/dart/json/xml/html/php/rtf/doc/docx/xls/xlsx/ppt/pptx/pdf и другие поддерживаемые.
- Кнопка сворачивания/раскрытия левой панели перенесена в область вкладок.
- Высота области вкладок уменьшается вместе с масштабом интерфейса.
