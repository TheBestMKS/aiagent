# ИИ Агент Flutter v1.5

Изменения:

- Программа переименована в «ИИ Агент».
- Добавлен повторный запрос к модели, если LM Studio вернул только reasoning_content и пустой content.
- Добавлено окно подробных логов с трассировкой запуска, проверки папок llama.cpp/models, запросов пользователя, HTTP-запросов к ИИ, ответов ИИ, tool-call и изменений файлов.
- Добавлена кнопка копирования всего диалога.
- Добавлено прикрепление файлов через встроенный проводник программы; вложения отображаются чипами с кнопкой удаления.
- Добавлена кнопка «Права» с режимами разрешений и режимами создания проекта.
- Добавлено окно настроек программы, включение/отключение логирования и проверки качества.
- Добавлен встроенный проводник файлов программы с копированием, вырезанием, вставкой, удалением и загрузкой файлов из доступных путей.
- Добавлены действия агента: project_map, append_file, replace_text, delete_path, copy_path, move_path, run_tests.
- Добавлены контекстные действия проектов: переименовать, изменить расположение, дублировать, удалить.
- В редакторе файлов текст выровнен сверху; добавлена галочка «Переносить по словам».

Важно: агент не содержит предметных шаблонов программ. Он исполняет команды, которые модель выдает через tool-call, и возвращает результаты модели для продолжения.
