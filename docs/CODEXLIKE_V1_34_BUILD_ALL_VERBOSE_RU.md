# AI Agent v1.35 — исправление silent build_all.bat

Исправлена ситуация, когда `build_all.bat` мог завершиться без видимого вывода.

Изменения:
- `build_all.bat` теперь печатает стартовую диагностику сразу, до проверки Flutter.
- Ошибки окружения Flutter больше не завершают скрипт молча: выводится понятная причина и выполняется `pause`.
- `_flutter_env.bat` ищет portable Flutter в `tooling\flutter\flutter\bin\flutter.bat`, а если его нет — пробует найти `flutter` в PATH.
- Все build-скрипты переведены на UTF-8 через `chcp 65001`, чтобы русские сообщения не превращались в кракозябры.
- Добавлен `diagnose_build_env.bat` для быстрой проверки окружения.
- dist-имена обновлены на v1.35.
