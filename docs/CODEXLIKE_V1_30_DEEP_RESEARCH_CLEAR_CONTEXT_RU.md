# AI Agent v1.30 — очистка контекста проекта и углублённый интернет-поиск

## Что добавлено

1. В контекстное меню проекта добавлена команда **«Очистить диалог и контекст»**.
   - Удаляются файлы сессии `.cppagent/sessions`.
   - Удаляется текущий план задачи `.cppagent/task_plan.md` и временные файлы контекста.
   - Файлы проекта, база знаний и логи не удаляются.

2. Добавлен инструмент агента `web_research`.
   - Выполняет первичный DuckDuckGo-поиск.
   - Открывает несколько найденных сайтов.
   - Извлекает ссылки, контакты, профили, полезные строки, внутренние и внешние переходы.
   - Формирует follow-up запросы по найденным доменам, профилям, контактам и строкам с признаками работы/связей.
   - Для поиска людей делает проверку точного ФИО.

3. Улучшен `web_fetch`.
   - Собирает больше ссылок со страницы.
   - Добавляет блоки `CONTACTS` и `USEFUL_LINES`.
   - Возвращает больше полезной информации для дальнейшего поиска.

4. Ссылки в диалоге открываются через меню:
   - **Открыть во вкладке Web**;
   - **Открыть с помощью внешнего браузера**;
   - **Копировать**.

5. Усилено правило чтения внешних папок устройства.
   - Для запросов вроде `Выведи содержимое файлов в папке "O:\..."` агент должен использовать `list_device_directory`, `read_device_text_file`, `read_device_folder_texts`, а не `run_command dir/type`.

## Проверка

ZIP проверен на целостность. Flutter SDK в среде сборки недоступен, поэтому `flutter analyze` и сборка не выполнялись.
