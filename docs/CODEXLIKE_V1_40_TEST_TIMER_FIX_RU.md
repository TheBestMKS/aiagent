# AI Agent v1.40 — исправление widget-теста и фоновых таймеров

## Что исправлено

- Добавлен тестовый режим `AiAgentApp(disableStartupTasks: true)`.
- В тестовом режиме не запускаются фоновые стартовые задачи приложения:
  - `controller.initialize()`;
  - отложенный запуск `_askLocalLlamaIfNeeded()`;
  - таймер ожидания локального llama-сервера.
- `test/widget_test.dart` теперь проверяет запуск приложения без фонового таймера, который раньше оставался pending после dispose.

## Причина ошибки v1.39

`flutter test` падал не из-за основной логики приложения, а из-за таймера `Future.delayed(800 ms)` в `_askLocalLlamaIfNeeded()`. В тестовом окружении Flutter запрещает завершать widget-тест, если после dispose остались активные таймеры.
