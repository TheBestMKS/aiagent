# AI Agent v1.46 — исправление реальных Dart-модулей

Исправлена первая версия настоящей модульной структуры v1.44.

Главные изменения:
- `askText` подключён через модуль `utils/html_utils.dart` в местах использования.
- `EmbeddedFilePickerDialog` подключён из `dialogs/settings_dialogs.dart` во вкладку чата.
- `ConsoleQuickAction` вынесен из UI-вкладки консоли в общий модуль `core/runtime_types.dart`, чтобы контроллер не зависел от вкладки.
- Тесты импортируют реальные модули приложения и утилит, а не ожидают экспортов из монолитного `main.dart`.
- `main.dart` остаётся короткой точкой входа и не превращается обратно в монолит.
