# v1.27 — Android UI/console, Web, dist, интернет/поиск/база знаний

## Главное

- Версия приложения обновлена до `v1.27` и отображается внизу окна.
- Добавлен масштаб интерфейса: настройки позволяют уменьшать/увеличивать текст и элементы через ползунок.
- На Android применяется автоматический компактный масштаб интерфейса.
- `run_command` теперь разрешён на Android и использует `/system/bin/sh -c`, чтобы вкладка «Консоль» работала как собственная оболочка в песочнице приложения.
- Android manifest patch добавляет `INTERNET`, `ACCESS_NETWORK_STATE`, `usesCleartextTraffic=true` и аппаратное ускорение для Web-вкладки.
- Встроенные build-скрипты копируют сборки в папку `dist` с именами, содержащими программу, версию, платформу и архитектуру.
- В конце build-скриптов добавлена `pause`; при запуске через `build_all.bat` вложенные паузы отключаются, пауза остаётся только в конце общего процесса.

## Новые инструменты агента

- `web_deep_fetch` — открывает страницу и переходит по внутренним ссылкам сайта на заданную глубину.
- `filesystem_search` — ищет реализации/исходники/архивы/документацию в файловой системе, если это разрешено настройками.
- `knowledge_store` — сохраняет полезную информацию в базе знаний.
- `knowledge_search` — ищет по базе знаний.

## Настройки

Добавлено:

- Масштаб интерфейса.
- Разрешить/запретить интернет-инструменты агента.
- Разрешить/запретить поиск по файловой системе компьютера.
- Окно базы знаний: просмотр, экспорт ZIP/JSONL, импорт ZIP/JSONL.

## Безопасность поиска людей

В системный prompt добавлено правило строгой проверки ФИО: если пользователь просит конкретного человека, например «Редин Максим Юрьевич», агент не должен подменять его «Максимом Игоревичем» или другим однофамильцем.

## Markdown

Строки с `#`, `##`, `###`, `####` теперь отображаются как заголовки. Если внутри заголовка есть `**жирный текст**`, жирное форматирование сохраняется.
