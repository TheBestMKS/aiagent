# Flutter v1.4

Добавлено:

- отдельное окно подробных логов выполнения;
- копирование текста лога;
- цикл выполнения большого количества действий модели;
- передача результатов инструментов обратно модели;
- исправленная правка AndroidManifest.xml;
- сборка Windows и Android в build_all.bat с копированием в distrib.

Важно: агент не содержит предметных шаблонов программ. Он выполняет действия, которые запрашивает модель через tool-call или кодовые блоки файлов.
