# V1.19 — интерфейс чата, консоль, Web, потоковые ответы

Изменения:

- В русской локали сообщения подписываются как `Пользователь:` и `Агент:` мелким шрифтом.
- У каждого сообщения отображается дата/время и контекстное меню с пунктом `Копировать`.
- Раскрытие code/action/diff блоков больше не вызывает принудительную прокрутку в конец диалога.
- Раскрытые блоки ограничены по высоте и имеют внутреннюю прокрутку, чтобы ползунок основного диалога не прыгал из-за огромных блоков.
- Кнопка настроек моделей перенесена в `Настройки программы` → `Модели`.
- Вкладки: `Чат`, `Файлы`, `Консоль`, `Web`.
- Вкладка `Консоль` поддерживает несколько вкладок, добавление через `+`, контекстное меню вкладок и правую панель быстрых команд.
- Вкладка `Web` поддерживает несколько вкладок, адресную строку, контекстное меню вкладок и правую панель быстрых адресов. Без внешних WebView-зависимостей отображается HTML/текст ответа.
- На узком экране левая панель проектов скрывается и открывается кнопкой.
- В настройках программы добавлен выбор языка и переопределение папки `Projects`.
- Встроенный выбор файлов/папок расширен режимом выбора папок.
- В профиле модели добавлен параметр `Потоковое получение ответа`, включённый по умолчанию.
- Для тяжёлых моделей таймаут ожидания ответа увеличен, а OpenAI-compatible потоковые SSE-ответы собираются в обычный ответ/tool_call.
- `build_all.bat` теперь пишет подробный лог в `build_logs\build_all_latest.log`, включая версию Flutter и свободное место на дисках.

Ограничение Web-вкладки: без добавления Flutter-плагинов WebView она показывает HTML/текст. Для полноценного браузера с JavaScript потребуется отдельная зависимость и платформенная настройка.
