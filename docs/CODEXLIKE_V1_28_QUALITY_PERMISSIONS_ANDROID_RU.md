# v1.28 — качество поиска, права проекта, Android-консоль и ссылки

Изменения:
- Приложение переименовано в **AI Agent** в заголовке окна и Android label.
- Версия обновлена до `v1.28`.
- Масштаб интерфейса теперь влияет не только на текст, но и на иконки кнопок.
- Права интернет-инструментов, поиска по устройству и доступа к файлам теперь относятся к текущему проекту и сохраняются в `.cppagent/project_permissions.json`.
- В настройках задаются права по умолчанию для новых проектов: `Запрашивать каждое действие`, `Автоматически с учетом сложности проекта`, интернет включен, поиск по устройству и доступ к файлам выключены.
- Добавлена проверка доступности прав и путей приложения в настройках.
- Android-консоль получила встроенный изолированный менеджер пакетов `pkg/apt/ai-pkg` для пространства `tools/android/packages`.
- Ссылки в сообщениях чата стали интерактивными: открыть во вкладке Web, открыть во внешнем браузере, копировать.
- Web-загрузчик использует User-Agent, редиректы и cleartext-настройки Android.
- Усилена проверка качества ответов: поисковые задачи должны использовать интернет-инструменты; для персон нужен поиск + чтение страниц; финальный ответ блокируется, если подменено отчество/ФИО.
- Агенту добавлены правила самопроверки: сверять точную задачу, источники и совпадение найденных фактов перед итогом.
