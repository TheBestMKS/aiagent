# AI Agent v1.33 — исправление ошибки сборки v1.32

Исправлена ошибка Dart-синтаксиса в `lib/main.dart` около блока извлечения изображений из HTML.

Причина: регулярные выражения были записаны в raw single-quoted строках с символами одинарной кавычки внутри (`r'...\'...'`). Dart завершал строку раньше времени и воспринимал остаток как список/операторы, из-за чего сборка падала ошибками `Can't find ] to match [` и `firstMatch isn't defined for List<dynamic>`.

Что изменено:
- проблемные RegExp переведены на raw double-quoted строки;
- версия обновлена до v1.33;
- build_all.bat больше не пишет ложную ошибку `flutter не является командой`, а вызывает переносимый `%FLUTTER%`;
- dist-имена обновлены на v1.33.
