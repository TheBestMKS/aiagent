# AI Agent v1.39 — исправление widget_test и тестового этапа сборки

## Что исправлено

В v1.38 сборка проходила анализ исходников, но падала на этапе `flutter test`, потому что `test/widget_test.dart` искал текст `AI Agent` через `find.text('AI Agent')`.
После изменений интерфейса название программы находится в левой панели и может не попасть в тестовый кадр/дерево текста так, как ожидал старый тест.

Тест обновлён: теперь он проверяет, что приложение запускается без исключения и создаёт `MaterialApp`.
Это соответствует назначению теста `App starts` и не привязано к конкретной строке интерфейса.

## Изменения

- `test/widget_test.dart` больше не ищет устаревший текст через `find.text('AI Agent')`.
- Добавлена проверка `tester.takeException() == null`.
- Добавлена проверка наличия `MaterialApp`.
- `run_tests.bat` запускает только тесты проекта: `flutter test test`.
- Версия обновлена до v1.39.
