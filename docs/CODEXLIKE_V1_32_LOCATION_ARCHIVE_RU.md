# AI Agent v1.32 — выбор расположения и архивирование внешних папок

## Что добавлено

- Добавлена кнопка **«Выбрать расположение»** рядом с кнопками **«Прикрепить файл»** и **«Копировать весь диалог»**.
- Кнопка открывает встроенное диалоговое окно выбора папки на устройстве.
- Выбранное расположение добавляется в контекст задачи как `SELECTED_LOCATION`, отображается отдельным чипом и может использоваться агентом в следующих действиях.
- Если расположение находится вне проекта, право **«Доступ к файлам устройства»** включается для текущего проекта после явного выбора пользователем.

## Исправление задачи с архивированием

- Добавлен инструмент `archive_device_children`.
- Инструмент создаёт отдельный ZIP-архив для каждого файла или подпапки внутри выбранной/внешней папки.
- По умолчанию архивы сохраняются в подпапку `_archives` внутри исходной папки.
- На Windows используется `PowerShell Compress-Archive`, на Linux/macOS/Android — `zip`, если он доступен.
- Агенту запрещено имитировать упаковку созданием пустых папок внутри проекта.

## Автоматическое исправление поведения агента

- Если пользователь пишет «упакуй каждую игру в отдельный архив», а ранее была прочитана или выбрана папка, приложение может выполнить `archive_device_children` напрямую, без нового бесполезного цикла модели.
- Последнее прочитанное/выбранное расположение устройства добавляется в system prompt, чтобы агент понимал фразы вроде «эту папку» или «каждую игру».
